# Диаграммы Mermaid

## 1. Блок-схема приготовления чая
```mermaid
flowchart TD
    start([Начало]) --> water[Кипятим воду]
    water --> check{Есть чай?}
    check -->|Да| brew[Завариваем чай]
    check -->|Нет| buy[Идем в магазин]
    buy --> get[Покупаем чай]
    get --> brew
    brew --> drink[Пьем чай]
    drink --> finish([Конец])
```

## 2. Диаграмма заказа такси
```mermaid
sequenceDiagram
    Клиент->>Приложение: Открываю приложение
    Приложение->>Клиент: Показываю карту
    Клиент->>Приложение: Ввожу адрес
    Приложение->>Сервер: Ищу водителя
    Сервер->>Водитель: Новый заказ
    Водитель->>Сервер: Беру заказ
    Сервер->>Приложение: Водитель найден
    Приложение->>Клиент: Такси едет
    Водитель->>Клиент: Приехал
    Клиент->>Водитель: Сажусь в авто
```

## 3. Диаграмма классов библиотеки
```mermaid
classDiagram
    class Книга {
        -название : String
        -автор : String
        -год : int
        +взять() void
        +вернуть() void
    }
    class Читатель {
        -имя : String
        -номер : String
        -книги[] : List
        +взятьКнигу() void
        +сдатьКнигу() void
    }
    class Библиотека {
        -книги[] : List
        -читатели[] : List
        +добавитьКнигу() void
        +зарегистрировать() void
    }

    Читатель "много" -- "много" Книга : берет
    Библиотека "1" *-- "много" Книга : хранит
    Библиотека "1" *-- "много" Читатель : обслуживает
```

## 8. ER-диаграмма сайта

erDiagram

        int id

    }

## 9. Блок-схема доставки еды
```mermaid
flowchart TD
    ЗашелВПриложение[Зашел в приложение] --> ВыбралЕду[Выбрал еду]
    ВыбралЕду --> Оформил[Оформил заказ]
    Оформил --> Оплатил[Оплатил]
    Оплатил --> Повар[Повар готовит]
    Повар --> Курьер[Курьер везет]
    Курьер --> Получил[Получил еду]
    Получил --> Съел[Съел]
    Съел --> Конец([Конец])
```
